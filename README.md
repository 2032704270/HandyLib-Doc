## 使用方法

#### 初始化jar
- 下载代码中repo文件夹的HandyLib-1.5.7.jar
- 或者使用maven引入依赖

```
<repositories>
   <repository>
      <id>handy-lib</id>
      <url>https://raw.githubusercontent.com/handy-git/HandLib-Doc/master/repo</url>
   </repository>
</repositories>

<dependency>
  <groupId>com.handy.lib</groupId>
  <artifactId>HandyLib</artifactId>
  <version>1.5.7</version>
</dependency>
```
